032    /**
033     * Main class for Simple XML Persistence Objects (POBJ).
034     *
035     * <h3>Overview</h3>
036     *
037     * <p>
038     * Instances model an in-memory "database" represented by a root Java object and the graph of other objects that
039     * it references. The object graph is backed by a persistent XML file, which is read at initialization time and
040     * re-written after each change.
041     *
042     * <p>
043     * Changes are applied "wholesale" to the entire object graph, and are serialized and atomic. In other words, the
044     * entire object graph is read from, and written to, this class by value. As a result, it is not possible to change
045     * only a portion of the "database". The entire object graph is read and written as one thing. Similarly, the
046     * persistent XML file is updated by writing out a new, temporary copy and renaming the copy onto the original,
047     * using {@link File#renameTo File.renameTo()} for atomicity (on systems that support it, e.g., UNIX variants).
048     *
049     * <h3>Update Details</h3>
050     *
051     * <p>
052     * When the object graph is updated, it must pass validation checks, and then the persistent XML file is updated and
053     * listener notifications are sent out. Support for delayed write-back of the persistent XML file is included: this
054     * allows modifications that occur in rapid succession to be consolidated into a single filesystem write operation.
055     *
056     * <p>
057     * Support for optimistic locking is included. There is a "current version" number which is incremented each
058     * time the object graph is updated; writes may optionally specify this number to ensure no intervening changes
059     * have occurred. If concurrent updates are expected, applications may choose to implement a 3-way merge algorithm
060     * of some kind to handle optimistic locking failures.
061     *
062     * <p>
063     * Instances can be configured to preserve one or more backup copies of the persistent file on systems that support
064     * hard links (requires <a href="https://github.com/twall/jna">JNA</a>; see {@link FileStreamRepository}).
065     * Set the {@link #getNumBackups numBackups} property to enable.
066     *
067     * <h3>"Out-of-band" Writes</h3>
068     *
069     * <p>
070     * When a non-zero {@linkplain #getCheckInterval check interval} is configured, instances support "out-of-band" writes
071     * to the XML persistent file by some other process. This can be handy in cases where the other process (perhaps hand edits)
072     * is updating the persistent file and you want to have a running process pick up the changes just as if
073     * {@link PersistentObject#setRoot setRoot()} had been invoked. In particular, instances will detect the appearance
074     * of a new persistent file after an instance has started without one. In all cases, persistent objects must properly validate.
075     *
076     * <p>
077     * A special case of this is effected when {@link PersistentObject#setRoot setRoot()} is never explicitly invoked
078     * by the application. Then some other process must be responsible for all database updates, and this class automatically
079     * picks them up, validates them, and send out notifications.
080     *
081     * <h3>Empty Starts and Stops</h3>
082     *
083     * <p>
084     * An "empty start" occurs when an instance is {@linkplain #start started} but the persistent XML file is either missing,
085     * does not validate, or cannot be read for some other reason. In such cases, the instance will start with no object graph,
086     * and {@link #getRoot} will return null. This situation will correct itself as soon as the object graph is written via
087     * {@link #setRoot setRoot()} or the persistent file appears (effecting an "out-of-band" update). At that time, {@linkplain
088     * #addListener listeners} will be notified for the first time, with {@link PersistentObjectEvent#getOldRoot} returning null.
089     *
090     * <p>
091     * Whether empty starts are allowed is determined by the {@link #isAllowEmptyStart allowEmptyStart} property (default
092     * {@code false}). When empty starts are disallowed and the persistent XML file cannot be successfully read,
093     * then {@link #start} will instead throw an immediate {@link PersistentObjectException}.
094     *
095     * <p>
096     * Similarly, "empty stops" are allowed when the {@link #isAllowEmptyStop allowEmptyStop} property is set to {@code true}
097     * (by default it is {@code false}). An "empty stop" occurs when a null value is passed to {@link #setRoot setRoot()}.
098     * Subsequent invocations of {@link #getRoot} will return {@code null}; however, the persistent file is <b>not</b>
099     * modified when null is passed to {@link #setRoot}. When empty stops are disallowed, then invoking
100     * {@link #setRoot} with a {@code null} object will result in an {@link IllegalArgumentException}.
101     *
102     * <p>
103     * Allowing empty starts and/or stops essentially creates an "unconfigured" state represented by a null root object.
104     * When empty starts and empty stops are both disallowed, there is no "unconfigured" state: {@link #getRoot} can be
105     * relied upon to always return a non-null, validated root object.
106     *
107     * <h3>Delegate Function</h3>
108     *
109     * <p>
110     * Instances must be configured with a {@link PersistentObjectDelegate} that knows how to validate the object graph
111     * and perform conversions to and from XML. See {@link PersistentObjectDelegate} and its implementations for details.
112     *
113     * <h3>Schema Changes</h3>
114     *
115     * <p>
116     * Like any database, the XML schema may evolve over time. The {@link PersistentObjectSchemaUpdater} class provides a simple
117     * way to apply and manage schema updates using XSLT transforms.
118     *
119     * @param <T> type of the root persistent object
120     * @see PersistentObjectDelegate
121     */

384        /**
385         * Start this instance. Does nothing if already started.
386         *
387         * @throws PersistentObjectException if an error occurs
388         */
389        public synchronized void start() {
390    
391            // Already started?
392            if (this.started)
393                return;
394    
395            // Sanity check
396            if (this.streamRepository == null)
397                throw new PersistentObjectException("no file configured");
398            if (this.delegate == null)
399                throw new PersistentObjectException("no delegate configured");
400    
401            // Create executor services
402            this.scheduledExecutor = Executors.newSingleThreadScheduledExecutor();
403            this.notifyExecutor = Executors.newSingleThreadExecutor();
404    
405            // Read file (if it exists)
406            this.log.info(this + ": starting");
407            if (this.getFile().exists()) {
408    
409                // Apply file
410                try {
411                    this.applyFile(this.getFile().lastModified(), true);
412                } catch (PersistentObjectException e) {
413                    if (!this.isAllowEmptyStart())
414                        throw e;
415                    this.log.warn("empty start: unable to load persistent file `" + this.getFile() + "': " + e);
416                }
417            } else {
418    
419                // Persistent file does not exist, so get default value from delegate
420                final T defaultValue = this.delegate.getDefaultValue();
421    
422                // If no default value, we have an empty start situation, otherwise apply the default value
423                if (defaultValue == null) {
424                    if (!this.isAllowEmptyStart()) {
425                        throw new PersistentObjectException("persistent file `" + this.getFile() + "' does not exist,"
426                          + " there is no default value, and empty starts are disallowed");
427                    }
428                    this.log.info(this + ": empty start: persistent file `" + this.getFile() + "' does not exist"
429                      + " and there is no default value");
430                } else {
431                    this.log.info(this + ": persistent file `" + this.getFile() + "' does not exist, applying default value");
432                    try {
433                        this.setRootInternal(defaultValue, 0, false, true);
434                    } catch (PersistentObjectException e) {                             // e.g., validation failure
435                        if (!this.isAllowEmptyStart())
436                            throw e;
437                        this.log.warn("empty start: unable to apply default value: " + e);
438                    }
439                }
440            }
441    
442            // Start checking the file
443            if (this.checkInterval > 0) {
444                this.scheduledExecutor.scheduleWithFixedDelay(new Runnable() {
445                    @Override
446                    public void run() {
447                        PersistentObject.this.checkFileTimeout();
448                    }
449                }, this.checkInterval, this.checkInterval, TimeUnit.MILLISECONDS);
450            }
451    
452            // Done
453            this.started = true;
454        }
455    
456        /**
457         * Stop this instance. Does nothing if already stopped.
458         *
459         * @throws PersistentObjectException if a delayed write back is pending and error occurs while performing the write
460         */
461        public synchronized void stop() {
462    
463            // Already stopped?
464            if (!this.started)
465                return;
466    
467            // Perform any lingering pending save now
468            if (this.cancelPendingWrite())
469                this.writeback();
470    
471            // Stop executor services
472            this.log.info(this + ": shutting down");
473            this.scheduledExecutor.shutdown();
474            this.notifyExecutor.shutdown();
475            this.awaitTermination(this.scheduledExecutor, "scheduledExecutor");
476            this.awaitTermination(this.notifyExecutor, "notifyExecutor");
477    
478            // Reset
479            this.scheduledExecutor = null;
480            this.notifyExecutor = null;
481            this.root = null;
482            this.sharedRoot = null;
483            this.writebackRoot = null;
484            this.version = 0;
485            this.timestamp = 0;
486            this.started = false;
487        }

490         * Atomically read the root object.
491         *
492         * <p>
493         * In the situation of an empty start or empty stop, this instance is "unconfigured" and null will be returned.
494         * This can only happen if {@link #setAllowEmptyStart setAllowEmptyStart(true)} or {@link #setAllowEmptyStop
495         * setAllowEmptyStop(true)} has been invoked.
496         *
497         * <p>
498         * This returns a deep copy of the current root object; any subsequent modifications are not written back.
499         * Use {@link #getSharedRoot} instead to avoid the cost of the deep copy at the risk of seeing modifications
500         * caused by other invokers.

503         * @throws IllegalStateException if this instance is not started
504         * @throws PersistentObjectException if an error occurs
505         */

507    
508            // Sanity check
509            if (!this.started)
510                throw new IllegalStateException("not started");
511

513            return this.root != null ? this.delegate.copy(this.root) : null;
514        }

566        /**
567         * Atomically update the root object.
568         *
569         * <p>
570         * The given object is deep-copied, the copy replaces the current root, and the new version number is returned.
571         * If there is no write delay configured, the new version is written to the underlying file immediately and
572         * a successful return from this method indicates the new root has been persisted. Otherwise, the write will
573         * occur at a later time in a separate thread.
574         * </p>
575         *
576         * <p>
577         * If {@code expectedVersion} is non-zero, then if the current version is not equal to it,
578         * a {@link PersistentObjectVersionException} exception is thrown. This mechanism
579         * can be used for optimistic locking.
580         * </p>
581         *
582         * <p>
583         * If empty stops are allowed, then {@code newRoot} may be null, in which case it replaces the
584         * current root and subsequent calls to {@link #getRoot} will return null. When a null
585         * {@code newRoot} is set, the persistent file is <b>not</b> modified.
586         * </p>
587         *
588         * <p>
589         * If the given root object is {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current
590         * root object, then no action is taken and the current (unchanged) version number is returned.
591         * </p>
592         *
593         * <p>
594         * After a successful change, any registered {@linkplain PersistentObjectListener listeners} are notified in a
595         * separate thread from the one that invoked this method.
596         * </p>
597         *
598         * @param newRoot new persistent object
599         * @param expectedVersion expected current version number, or zero to ignore the current version number
600         * @return the new current version number (unchanged if {@code newRoot} is
601         *  {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current root)
602         * @throws IllegalArgumentException if {@code newRoot} is null and empty stops are disallowed
603         * @throws IllegalArgumentException if {@code expectedVersion} is negative
604         * @throws IllegalStateException if this instance is not started
605         * @throws PersistentObjectException if an error occurs
606         * @throws PersistentObjectVersionException if {@code expectedVersion} is non-zero and not equal to the current version
607         * @throws PersistentObjectValidationException if the new root has validation errors
608         */
609        public final synchronized long setRoot(T newRoot, long expectedVersion) {
610            return this.setRootInternal(newRoot, expectedVersion, false, false);
611        }

613        @SuppressWarnings("unchecked")
614        private synchronized long setRootInternal(T newRoot, long expectedVersion, boolean readingFile, boolean allowNotStarted) {
615    
616            // Sanity check
617            if (newRoot == null && !this.isAllowEmptyStop())
618                throw new IllegalArgumentException("newRoot is null but empty stops are not enabled");
619            if (!this.started && !allowNotStarted)
620                throw new IllegalStateException("not started");
621            if (expectedVersion < 0)
622                throw new IllegalStateException("negative expectedVersion");
623    
624            // Check version number
625            if (expectedVersion != 0 && this.version != expectedVersion)
626                throw new PersistentObjectVersionException(this.version, expectedVersion);
627    
628            // Check for sameness
629            if (this.root == null && newRoot == null)
630                return this.version;
631            if (this.root != null && newRoot != null && this.delegate.isSameGraph(this.root, newRoot))
632                return this.version;
633    
634            // Validate the new root
635            if (newRoot != null) {
636                Set<ConstraintViolation<T>> violations = this.delegate.validate(newRoot);
637                if (!violations.isEmpty())
638                    throw new PersistentObjectValidationException((Set<ConstraintViolation<?>>)(Object)violations);
639            }
640    
641            // Do the update
642            final T oldRoot = this.root;
643            this.root = newRoot != null ? this.delegate.copy(newRoot) : null;
644            this.version++;
645            this.sharedRoot = null;
646    
647            // Perform write-back, either now or later
648            if (!readingFile && this.root != null) {
649                this.writebackRoot = this.root;
650                if (this.writeDelay == 0)
651                    this.writeback();
652                else if (this.pendingWrite == null) {
653                    this.pendingWrite = this.scheduledExecutor.schedule(new Runnable() {
654                        @Override
655                        public void run() {
656                            PersistentObject.this.writeTimeout();
657                        }
658                    }, this.writeDelay, TimeUnit.MILLISECONDS);
659                }
660            }
661    
662            // Notify listeners
663            this.notifyListeners(this.version, oldRoot, newRoot);
664    
665            // Done
666            return this.version;
667        }

685        /**
686         * Check the persistent file for an "out-of-band" update.
687         *
688         * <p>
689         * If the persistent file has a newer timestamp than the timestamp of the most recently read
690         * or written version, then it will be read and applied to this instance.
691         *
692         * @throws IllegalStateException if this instance is not started
693         * @throws PersistentObjectException if an error occurs
694         */
695        public synchronized void checkFile() {
696    
697            // Sanity check
698            if (!this.started)
699                throw new IllegalStateException("not started");
700    
701            // Get file timestamp
702            long fileTime = this.getFile().lastModified();
703            if (fileTime == 0)
704                return;
705    
706            // Check whether file has newly appeared or just been updated
707            if (this.timestamp != 0 && fileTime <= this.timestamp)
708                return;
709    
710            // Read new file
711            this.log.info(this + ": detected out-of-band update of persistent file `" + this.getFile() + "'");
712            this.applyFile(fileTime, false);
713        }

716         * Add a listener to be notified each time the object graph changes.
717         *

719         * Listeners are notified in a separate thread from the one that caused the root object to change.
720         *
721         * @throws IllegalArgumentException if {@code listener} is null
722         */

725                throw new IllegalArgumentException("null listener");

748        /**
749         * Read the persistent file.
750         *
751         * @throws PersistentObjectException if an error occurs
752         */
753        protected T read() {
754    
755            // Open file
756            this.log.info(this + ": reading persistent file `" + this.getFile() + "'");
757            BufferedInputStream input;
758            try {
759                input = new BufferedInputStream(this.streamRepository.getInputStream());
760            } catch (IOException e) {
761                throw new PersistentObjectException("error opening persistent file", e);
762            }
763    
764            // Parse XML
765            T obj;
766            try {
767                StreamSource source = new StreamSource(input);
768                source.setSystemId(this.getFile());
769                try {
770                    obj = this.delegate.deserialize(source);
771                } catch (IOException e) {
772                    throw new PersistentObjectException("error reading persistent file", e);
773                }
774            } finally {
775                try {
776                    input.close();
777                } catch (IOException e) {
778                    // ignore
779                }
780            }
781    
782            // Check result
783            if (obj == null)
784                throw new PersistentObjectException("null object returned by delegate.deserialize()");
785    
786            // Done
787            return obj;
788        }

790        /**
791         * Write the persistent file and rotate any backups.
792         *
793         * <p>
794         * A temporary file is created in the same directory and then renamed to provide for an atomic update
795         * (on supporting operating systems).
796         *
797         * @throws IllegalArgumentException if {@code obj} is null
798         * @throws PersistentObjectException if an error occurs
799         */
800        protected final synchronized void write(T obj) {
801    
802            // Sanity check
803            if (obj == null)
804                throw new IllegalArgumentException("null obj");
805    
806            // Open atomic update output and buffer it
807            AtomicUpdateFileOutputStream updateOutput;
808            try {
809                updateOutput = this.streamRepository.getOutputStream();
810            } catch (IOException e) {
811                throw new PersistentObjectException("error creating temporary file", e);
812            }
813            BufferedOutputStream output = new BufferedOutputStream(updateOutput);
814    
815            // Serialize to XML
816            try {
817    
818                // Set up XML result
819                StreamResult result = new StreamResult(output);
820                result.setSystemId(updateOutput.getTempFile());
821    
822                // Serialize root object
823                try {
824                    this.delegate.serialize(obj, result);
825                } catch (IOException e) {
826                    throw new PersistentObjectException("error writing persistent file", e);
827                }
828    
829                // Commit output
830                try {
831                    output.close();
832                } catch (IOException e) {
833                    throw new PersistentObjectException("error closing temporary file", e);
834                }
835    
836                // Success
837                output = null;
838            } finally {
839                if (output != null)
840                    updateOutput.cancel();
841            }
842    
843            // Update file timestamp
844            this.timestamp = this.streamRepository.getFileTimestamp();
845        }

847        /**
848         * Notify listeners of a change in value.
849         *
850         * @param newVersion the version number associated with the new root
851         */
852        protected void notifyListeners(long newVersion, T oldRoot, T newRoot) {
853    
854            // Snapshot listeners
855            final ArrayList<PersistentObjectListener<T>> listenersCopy = new ArrayList<PersistentObjectListener<T>>();
856            synchronized (this.listeners) {
857                listenersCopy.addAll(this.listeners);
858            }
859    
860            // Notify them
861            final PersistentObjectEvent<T> event = new PersistentObjectEvent<T>(this, newVersion, oldRoot, newRoot);
862            this.notifyExecutor.submit(new Runnable() {
863                @Override
864                public void run() {
865                    PersistentObject.this.doNotifyListeners(listenersCopy, event);
866                }
867            });
868        }

870        // Read the persistent file and apply it
871        private synchronized void applyFile(long newTimestamp, boolean allowNotStarted) {
872            this.cancelPendingWrite();
873            this.timestamp = newTimestamp;                              // update timestamp even if update fails to avoid loops
874            this.setRootInternal(this.read(), 0, true, allowNotStarted);
875        }

877        // Handle a write-back timeout
878        private synchronized void writeTimeout() {
879    
880            // Check for cancel race
881            if (this.pendingWrite == null)
882                return;
883            this.pendingWrite = null;
884    
885            // Write it
886            try {
887                this.writeback();
888            } catch (ThreadDeath t) {
889                throw t;
890            } catch (Throwable t) {
891                this.delegate.handleWritebackException(this, t);
892            }
893        }
894    
895        // Handle a check file timeout
896        private synchronized void checkFileTimeout() {
897    
898            // Handle race condition
899            if (!this.started)
900                return;
901    
902            // Check file
903            try {
904                this.checkFile();
905            } catch (ThreadDeath t) {
906                throw t;
907            } catch (Throwable t) {
908                this.log.error(this + ": error attempting to apply out-of-band update", t);
909            }
910        }
911    
912        // Write back root to persistent file
913        private synchronized void writeback() {
914            T objectToWrite = this.writebackRoot;
915            assert objectToWrite != null;
916            this.writebackRoot = null;
917            this.write(objectToWrite);
918        }
919    
920        // Cancel a pending write and return true if there was one
921        private synchronized boolean cancelPendingWrite() {
922            if (this.pendingWrite == null)
923                return false;
924            this.pendingWrite.cancel(false);
925            this.pendingWrite = null;
926            return true;
927        }
928    
929        // Notify listeners. This is invoked in a separate thread.
930        private void doNotifyListeners(ArrayList<PersistentObjectListener<T>> list, PersistentObjectEvent<T> event) {
931            for (PersistentObjectListener<T> listener : list) {
932                try {
933                    listener.handleEvent(event);
934                } catch (ThreadDeath t) {
935                    throw t;
936                } catch (Throwable t) {
937                    this.log.error(this + ": error notifying listener " + listener, t);
938                }
939            }
940        }

942        // Wait for an ExecutorService to completely shut down
943        private void awaitTermination(ExecutorService executor, String name) {
944            boolean shutdown = false;
945            try {
946                shutdown = executor.awaitTermination(EXECUTOR_SHUTDOWN_TIMEOUT, TimeUnit.MILLISECONDS);
947            } catch (InterruptedException e) {
948                this.log.warn(this + ": interrupted while awaiting " + name + " termination");
949            }
950            if (!shutdown)
951                this.log.warn(this + ": failed to completely shut down " + name);
952        }
