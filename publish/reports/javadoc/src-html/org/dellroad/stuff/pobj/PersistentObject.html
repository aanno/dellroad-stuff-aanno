032    /**
033     * Main class for Simple XML Persistence Objects (POBJ).
034     *
035     * <h3>Overview</h3>
036     *
037     * <p>
038     * Instances model an in-memory "database" represented by a root Java object and the graph of other objects that
039     * it references. The object graph is backed by a persistent XML file, which is read at initialization time and
040     * re-written after each change.
041     *
042     * <p>
043     * Changes are applied "wholesale" to the entire object graph, and are serialized and atomic. In other words, the
044     * entire object graph is read from, and written to, this class by value. As a result, it is not possible to change
045     * only a portion of the "database". The entire object graph is read and written as one thing. Similarly, the
046     * persistent XML file is updated by writing out a new, temporary copy and renaming the copy onto the original,
047     * using {@link File#renameTo File.renameTo()} for atomicity (on systems that support it, e.g., UNIX variants).
048     *
049     * <h3>Update Details</h3>
050     *
051     * <p>
052     * When the object graph is updated, it must pass validation checks, and then the persistent XML file is updated and
053     * listener notifications are sent out. Support for delayed write-back of the persistent XML file is included: this
054     * allows modifications that occur in rapid succession to be consolidated into a single filesystem write operation.
055     *
056     * <p>
057     * Support for optimistic locking is included. There is a "current version" number which is incremented each
058     * time the object graph is updated; writes may optionally specify this number to ensure no intervening changes
059     * have occurred. If concurrent updates are expected, applications may choose to implement a 3-way merge algorithm
060     * of some kind to handle optimistic locking failures.
061     *
062     * <p>
063     * Instances can be configured to preserve one or more backup copies of the persistent file on systems that support
064     * hard links (requires <a href="https://github.com/twall/jna">JNA</a>; see {@link FileStreamRepository}).
065     * Set the {@link #getNumBackups numBackups} property to enable.
066     *
067     * <h3>"Out-of-band" Writes</h3>
068     *
069     * <p>
070     * When a non-zero {@linkplain #getCheckInterval check interval} is configured, instances support "out-of-band" writes
071     * to the XML persistent file by some other process. This can be handy in cases where the other process (perhaps hand edits)
072     * is updating the persistent file and you want to have a running process pick up the changes just as if
073     * {@link PersistentObject#setRoot setRoot()} had been invoked. In particular, instances will detect the appearance
074     * of a new persistent file after an instance has started without one. In all cases, persistent objects must properly validate.
075     *
076     * <p>
077     * A special case of this is effected when {@link PersistentObject#setRoot setRoot()} is never explicitly invoked
078     * by the application. Then some other process must be responsible for all database updates, and this class automatically
079     * picks them up, validates them, and send out notifications.
080     *
081     * <h3>Empty Starts and Stops</h3>
082     *
083     * <p>
084     * An "empty start" occurs when an instance is {@linkplain #start started} but the persistent XML file is either missing,
085     * does not validate, or cannot be read for some other reason. In such cases, the instance will start with no object graph,
086     * and {@link #getRoot} will return null. This situation will correct itself as soon as the object graph is written via
087     * {@link #setRoot setRoot()} or the persistent file appears (effecting an "out-of-band" update). At that time, {@linkplain
088     * #addListener listeners} will be notified for the first time, with {@link PersistentObjectEvent#getOldRoot} returning null.
089     *
090     * <p>
091     * Whether empty starts are allowed is determined by the {@link #isAllowEmptyStart allowEmptyStart} property (default
092     * {@code false}). When empty starts are disallowed and the persistent XML file cannot be successfully read,
093     * then {@link #start} will instead throw an immediate {@link PersistentObjectException}.
094     *
095     * <p>
096     * Similarly, "empty stops" are allowed when the {@link #isAllowEmptyStop allowEmptyStop} property is set to {@code true}
097     * (by default it is {@code false}). An "empty stop" occurs when a null value is passed to {@link #setRoot setRoot()}.
098     * Subsequent invocations of {@link #getRoot} will return {@code null}; however, the persistent file is <b>not</b>
099     * modified when null is passed to {@link #setRoot}. When empty stops are disallowed, then invoking
100     * {@link #setRoot} with a {@code null} object will result in an {@link IllegalArgumentException}.
101     *
102     * <p>
103     * Allowing empty starts and/or stops essentially creates an "unconfigured" state represented by a null root object.
104     * When empty starts and empty stops are both disallowed, there is no "unconfigured" state: {@link #getRoot} can be
105     * relied upon to always return a non-null, validated root object.
106     *
107     * <h3>Delegate Function</h3>
108     *
109     * <p>
110     * Instances must be configured with a {@link PersistentObjectDelegate} that knows how to validate the object graph
111     * and perform conversions to and from XML. See {@link PersistentObjectDelegate} and its implementations for details.
112     *
113     * <h3>Schema Changes</h3>
114     *
115     * <p>
116     * Like any database, the XML schema may evolve over time. The {@link PersistentObjectSchemaUpdater} class provides a simple
117     * way to apply and manage schema updates using XSLT transforms.
118     *
119     * @param <T> type of the root persistent object
120     * @see PersistentObjectDelegate
121     */

302        /**
303         * Start this instance. Does nothing if already started.
304         *
305         * @throws PersistentObjectException if an error occurs
306         */
307        public synchronized void start() {
308    
309            // Already started?
310            if (this.started)
311                return;
312    
313            // Create executor services
314            this.scheduledExecutor = Executors.newSingleThreadScheduledExecutor();
315            this.notifyExecutor = Executors.newSingleThreadExecutor();
316    
317            // Read file (if it exists)
318            this.log.info(this + ": starting");
319            if (this.getPersistentFile().exists()) {
320                try {
321                    this.applyFile(this.getPersistentFile().lastModified());
322                } catch (PersistentObjectException e) {
323                    if (!this.isAllowEmptyStart())
324                        throw e;
325                    this.log.warn("empty start: unable to load persistent file `" + this.getPersistentFile() + "': " + e);
326                }
327            } else {
328                if (!this.isAllowEmptyStart())
329                    throw new PersistentObjectException("persistent file `" + this.getPersistentFile() + "' does not exist");
330                this.log.info(this + ": empty start: persistent file `" + this.getPersistentFile() + "' does not exist");
331            }
332    
333            // Start checking the file
334            if (this.checkInterval > 0) {
335                this.scheduledExecutor.scheduleWithFixedDelay(new Runnable() {
336                    @Override
337                    public void run() {
338                        PersistentObject.this.checkFileTimeout();
339                    }
340                }, this.checkInterval, this.checkInterval, TimeUnit.MILLISECONDS);
341            }
342    
343            // Done
344            this.started = true;
345        }
346    
347        /**
348         * Stop this instance. Does nothing if already stopped.
349         *
350         * @throws PersistentObjectException if a delayed write back is pending and error occurs while performing the write
351         */
352        public synchronized void stop() {
353    
354            // Already stopped?
355            if (!this.started)
356                return;
357    
358            // Perform any lingering pending save now
359            if (this.cancelPendingWrite())
360                this.writeback();
361    
362            // Stop executor services
363            this.log.info(this + ": shutting down");
364            this.scheduledExecutor.shutdown();
365            this.notifyExecutor.shutdown();
366            this.awaitTermination(this.scheduledExecutor, "scheduledExecutor");
367            this.awaitTermination(this.notifyExecutor, "notifyExecutor");
368    
369            // Reset
370            this.scheduledExecutor = null;
371            this.notifyExecutor = null;
372            this.root = null;
373            this.sharedRoot = null;
374            this.writebackRoot = null;
375            this.version = 0;
376            this.timestamp = 0;
377            this.started = false;
378        }

380        /**
381         * Atomically read the root object.
382         *
383         * <p>
384         * In the situation of an empty start or empty stop, this instance is "unconfigured" and null will be returned.
385         * This can only happen if {@link #setAllowEmptyStart setAllowEmptyStart(true)} or {@link #setAllowEmptyStop
386         * setAllowEmptyStop(true)} has been invoked.
387         *
388         * <p>
389         * This returns a deep copy of the current root object; any subsequent modifications are not written back.
390         * Use {@link #getSharedRoot} instead to avoid the cost of the deep copy at the risk of seeing modifications
391         * caused by other invokers.
392         *
393         * @return the current root instance, or null during an empty start or empty stop
394         * @throws IllegalStateException if this instance is not started
395         * @throws PersistentObjectException if an error occurs
396         */
397        public synchronized T getRoot() {
398    
399            // Sanity check
400            if (!this.started)
401                throw new IllegalStateException("not started");
402    
403            // Copy root
404            return this.root != null ? this.delegate.copy(this.root) : null;
405        }

409         *
410         * <p>
411         * This returns a copy of the root object, but it returns the same copy each time until the next change.
412         * This method is more efficient than {@link #getRoot}, but all callers must agree not to modify the returned object
413         * or any object in its graph of references.
414         *
415         * @return shared copy of the root instance, or null during an empty start or empty stop
416         * @throws IllegalStateException if this instance is not started
417         */
418        public synchronized T getSharedRoot() {
419            if (this.sharedRoot == null)
420                this.sharedRoot = this.getRoot();
421            return this.sharedRoot;
422        }

424        /**
425         * Atomically update the root object.
426         *
427         * <p>
428         * The given object is deep-copied and the copy replaces the current root.
429         *
430         * <p>
431         * If {@code expectedVersion} is non-zero, then if the current version is not equal to it,
432         * a {@link PersistentObjectVersionException} exception is thrown. This mechanism
433         * can be used for optimistic locking.
434         *
435         * <p>
436         * If empty stops are allowed, then {@code newRoot} may be null, in which case it replaces the
437         * current root and subsequent calls to {@link #getRoot} will return null. When a null
438         * {@code newRoot} is set, the persistent file is <b>not</b> modified.
439         *
440         * <p>
441         * If the given root object is {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current
442         * root object, then no action is taken.
443         *
444         * <p>
445         * After a successful change, any registered {@linkplain PersistentObjectListener listeners} are notified in a
446         * separate thread from the one that invoked this method.
447         *
448         * @param newRoot new persistent object
449         * @param expectedVersion expected current version number, or zero to ignore the current version number
450         * @throws IllegalArgumentException if {@code newRoot} is null and empty stops are disallowed
451         * @throws IllegalArgumentException if {@code version} is negative
452         * @throws IllegalStateException if this instance is not started
453         * @throws PersistentObjectException if an error occurs
454         * @throws PersistentObjectVersionException if {@code version} is non-zero and not equal to the current version
455         * @throws PersistentObjectValidationException if the new root has validation errors
456         */
457        public final synchronized void setRoot(T newRoot, long expectedVersion) {
458            this.setRootInternal(newRoot, expectedVersion, false);
459        }
460    
461        @SuppressWarnings("unchecked")
462        private synchronized void setRootInternal(T newRoot, long expectedVersion, boolean readingFile) {
463    
464            // Sanity check
465            if (newRoot == null && !this.isAllowEmptyStop())
466                throw new IllegalArgumentException("newRoot is null but empty stops are not enabled");
467            if (!this.started && !readingFile)
468                throw new IllegalStateException("not started");
469            if (expectedVersion < 0)
470                throw new IllegalStateException("negative expectedVersion");
471    
472            // Check version number
473            if (expectedVersion != 0 && this.version != expectedVersion)
474                throw new PersistentObjectVersionException(this.version, expectedVersion);
475    
476            // Check for sameness
477            if (this.root == null && newRoot == null)
478                return;
479            if (this.root != null && newRoot != null && this.delegate.isSameGraph(this.root, newRoot))
480                return;
481    
482            // Validate the new root
483            if (newRoot != null) {
484                Set<ConstraintViolation<T>> violations = this.delegate.validate(newRoot);
485                if (!violations.isEmpty())
486                    throw new PersistentObjectValidationException((Set<ConstraintViolation<?>>)(Object)violations);
487            }
488    
489            // Do the update
490            final T oldRoot = this.root;
491            this.root = newRoot != null ? this.delegate.copy(newRoot) : null;
492            this.version++;
493            this.sharedRoot = null;
494    
495            // Perform write-back, either now or later
496            if (!readingFile && this.root != null) {
497                this.writebackRoot = this.root;
498                if (this.writeDelay == 0)
499                    this.writeback();
500                else if (this.pendingWrite == null) {
501                    this.pendingWrite = this.scheduledExecutor.schedule(new Runnable() {
502                        @Override
503                        public void run() {
504                            PersistentObject.this.writeTimeout();
505                        }
506                    }, this.writeDelay, TimeUnit.MILLISECONDS);
507                }
508            }
509    
510            // Notify listeners
511            this.notifyListeners(this.version, oldRoot, newRoot);
512        }

530        /**
531         * Check the persistent file for an "out-of-band" update.
532         *
533         * <p>
534         * If the persistent file has a newer timestamp than the timestamp of the most recently read
535         * or written version, then it will be read and applied to this instance.
536         *
537         * @throws IllegalStateException if this instance is not started
538         * @throws PersistentObjectException if an error occurs
539         */
540        public synchronized void checkFile() {
541    
542            // Sanity check
543            if (!this.started)
544                throw new IllegalStateException("not started");
545    
546            // Get file timestamp
547            long fileTime = this.getPersistentFile().lastModified();
548            if (fileTime == 0)
549                return;
550    
551            // Check whether file has newly appeared or just been updated
552            if (this.timestamp != 0 && fileTime <= this.timestamp)
553                return;
554    
555            // Read new file
556            this.log.info(this + ": detected out-of-band update of persistent file `" + this.getPersistentFile() + "'");
557            this.applyFile(fileTime);
558        }

600        /**
601         * Read the persistent file.
602         *
603         * @throws PersistentObjectException if an error occurs
604         */
605        protected T read() {
606    
607            // Open file
608            this.log.info(this + ": reading persistent file `" + this.getPersistentFile() + "'");
609            BufferedInputStream input;
610            try {
611                input = new BufferedInputStream(this.streamRepository.getInputStream());
612            } catch (IOException e) {
613                throw new PersistentObjectException("error opening persistent file", e);
614            }
615    
616            // Parse XML
617            T obj;
618            try {
619                StreamSource source = new StreamSource(input);
620                source.setSystemId(this.getPersistentFile());
621                try {
622                    obj = this.delegate.deserialize(source);
623                } catch (IOException e) {
624                    throw new PersistentObjectException("error reading persistent file", e);
625                }
626            } finally {
627                try {
628                    input.close();
629                } catch (IOException e) {
630                    // ignore
631                }
632            }
633    
634            // Check result
635            if (obj == null)
636                throw new PersistentObjectException("null object returned by delegate.deserialize()");
637    
638            // Done
639            return obj;
640        }

642        /**
643         * Write the persistent file and rotate any backups.
644         *
645         * <p>
646         * A temporary file is created in the same directory and then renamed to provide for an atomic update
647         * (on supporting operating systems).
648         *
649         * @throws IllegalArgumentException if {@code obj} is null
650         * @throws PersistentObjectException if an error occurs
651         */
652        protected final synchronized void write(T obj) {
653    
654            // Sanity check
655            if (obj == null)
656                throw new IllegalArgumentException("null obj");
657    
658            // Open atomic update output and buffer it
659            AtomicUpdateFileOutputStream updateOutput;
660            try {
661                updateOutput = this.streamRepository.getOutputStream();
662            } catch (IOException e) {
663                throw new PersistentObjectException("error creating temporary file", e);
664            }
665            BufferedOutputStream output = new BufferedOutputStream(updateOutput);
666    
667            // Serialize to XML
668            try {
669    
670                // Set up XML result
671                StreamResult result = new StreamResult(output);
672                result.setSystemId(updateOutput.getTempFile());
673    
674                // Serialize root object
675                try {
676                    this.delegate.serialize(obj, result);
677                } catch (IOException e) {
678                    throw new PersistentObjectException("error writing persistent file", e);
679                }
680    
681                // Commit output
682                try {
683                    output.close();
684                } catch (IOException e) {
685                    throw new PersistentObjectException("error closing temporary file", e);
686                }
687    
688                // Success
689                output = null;
690            } finally {
691                if (output != null)
692                    updateOutput.cancel();
693            }
694    
695            // Update file timestamp
696            this.timestamp = this.streamRepository.getFileTimestamp();
697        }
698    
699        /**
700         * Notify listeners of a change in value.
701         *
702         * @param newVersion the version number associated with the new root
703         */
704        protected void notifyListeners(long newVersion, T oldRoot, T newRoot) {
705    
706            // Snapshot listeners
707            final ArrayList<PersistentObjectListener<T>> listenersCopy = new ArrayList<PersistentObjectListener<T>>();
708            synchronized (this.listeners) {
709                listenersCopy.addAll(this.listeners);
710            }
711    
712            // Notify them
713            final PersistentObjectEvent<T> event = new PersistentObjectEvent<T>(this, newVersion, oldRoot, newRoot);
714            this.notifyExecutor.submit(new Runnable() {
715                @Override
716                public void run() {
717                    PersistentObject.this.doNotifyListeners(listenersCopy, event);
718                }
719            });
720        }
721    
722        // Read the persistent file and apply it
723        private synchronized void applyFile(long newTimestamp) {
724            this.cancelPendingWrite();
725            this.timestamp = newTimestamp;                              // update timestamp even if update fails to avoid loops
726            this.setRootInternal(this.read(), 0, true);
727        }
728    
729        // Handle a write-back timeout
730        private synchronized void writeTimeout() {
731    
732            // Check for cancel race
733            if (this.pendingWrite == null)
734                return;
735            this.pendingWrite = null;
736    
737            // Write it
738            try {
739                this.writeback();
740            } catch (ThreadDeath t) {
741                throw t;
742            } catch (Throwable t) {
743                this.delegate.handleWritebackException(this, t);
744            }
745        }
746    
747        // Handle a check file timeout
748        private synchronized void checkFileTimeout() {
749    
750            // Handle race condition
751            if (!this.started)
752                return;
753    
754            // Check file
755            try {
756                this.checkFile();
757            } catch (ThreadDeath t) {
758                throw t;
759            } catch (Throwable t) {
760                this.log.error(this + ": error attempting to apply out-of-band update", t);
761            }
762        }
763    
764        // Write back root to persistent file
765        private synchronized void writeback() {
766            T objectToWrite = this.writebackRoot;
767            assert objectToWrite != null;
768            this.writebackRoot = null;
769            this.write(objectToWrite);
770        }
771    
772        // Cancel a pending write and return true if there was one
773        private synchronized boolean cancelPendingWrite() {
774            if (this.pendingWrite == null)
775                return false;
776            this.pendingWrite.cancel(false);
777            this.pendingWrite = null;
778            return true;
779        }
780    
781        // Notify listeners. This is invoked in a separate thread.
782        private void doNotifyListeners(ArrayList<PersistentObjectListener<T>> list, PersistentObjectEvent<T> event) {
783            for (PersistentObjectListener<T> listener : list) {
784                try {
785                    listener.handleEvent(event);
786                } catch (ThreadDeath t) {
787                    throw t;
788                } catch (Throwable t) {
789                    this.log.error(this + ": error notifying listener " + listener, t);
790                }
791            }
792        }
793    
794        // Wait for an ExecutorService to completely shut down
795        private void awaitTermination(ExecutorService executor, String name) {
796            boolean shutdown = false;
797            try {
798                shutdown = executor.awaitTermination(EXECUTOR_SHUTDOWN_TIMEOUT, TimeUnit.MILLISECONDS);
799            } catch (InterruptedException e) {
800                this.log.warn(this + ": interrupted while awaiting " + name + " termination");
801            }
802            if (!shutdown)
803                this.log.warn(this + ": failed to completely shut down " + name);
804        }
