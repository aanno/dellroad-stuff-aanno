034    /**
035     * Main class for Simple XML Persistence Objects (POBJ).
036     *
037     * <h3>Overview</h3>
038     *
039     * <p>
040     * Instances model a fully validated, in-memory "database" represented by a root Java object and the graph of other objects
041     * that it references. The object graph is backed by a persistent XML file, which is read at initialization time and
042     * re-written after each change.
043     *
044     * <p>
045     * Changes are applied "wholesale" to the entire object graph, and are serialized and atomic. In other words, the
046     * entire object graph is read from, and written to, this class by value. As a result, it is not possible to change
047     * only a portion of the "database". The entire object graph is read and written as one thing. Similarly, the
048     * persistent XML file is updated by writing out a new, temporary copy and renaming the copy onto the original,
049     * using {@link File#renameTo File.renameTo()} for atomicity (on systems that support it, e.g., UNIX variants).
050     *
051     * <h3>Validation</h3>
052     *
053     * Validation is peformed in Java (not in XML via XSD) and defined by the provided delegate. This class guarantees
054     * that only a valid root Java object can be set, read or written. An invalid XML file on disk will generate an error
055     * (or be ignored; see "Empty Starts" below); setting an invalid Java root object via {@link #setRoot setRoot()}
056     * with throw an exception.
057     *
058     * <h3>Update Details</h3>
059     *
060     * <p>
061     * When the object graph is updated, it must pass validation checks, and then the persistent XML file is updated and
062     * listener notifications are sent out. Listeners are always notified in a separate thread from the one that invoked
063     * {@link #setRoot setRoot()}. Support for delayed write-back of the persistent XML file is included: this
064     * allows modifications that occur in rapid succession to be consolidated into a single filesystem write operation.
065     *
066     * <p>
067     * Support for optimistic locking is included. There is a "current version" number which is incremented each
068     * time the object graph is updated; writes may optionally specify this number to ensure no intervening changes
069     * have occurred. If concurrent updates are expected, applications may choose to implement a 3-way merge algorithm
070     * of some kind to handle optimistic locking failures.
071     *
072     * <p>
073     * Instances can be configured to automatically preserve one or more backup copies of the persistent file on systems that
074     * support hard links (requires <a href="https://github.com/twall/jna">JNA</a>; see {@link FileStreamRepository}).
075     * Set the {@link #getNumBackups numBackups} property to enable.
076     *
077     * <h3>"Out-of-band" Writes</h3>
078     *
079     * <p>
080     * When a non-zero {@linkplain #getCheckInterval check interval} is configured, instances support "out-of-band" writes
081     * directly to the XML persistent file by some other process. This can be handy in cases where the other process (perhaps hand
082     * edits) is updating the persistent file and you want to have the running Java process pick up the changes just as if
083     * {@link PersistentObject#setRoot setRoot()} had been invoked. In particular, instances will detect the appearance
084     * of a new persistent file after an instance has started without one. In all cases, persistent objects must properly validate.
085     * To avoid reading a partial file the external process should write the file atomically by creating a temporary file and
086     * renaming it; however, this race window is small and in any case the problem is self-correcting because a partially
087     * written XML file will not validate, and so it will be ignored and retried after another {@linkplain #getCheckInterval
088     * check interval} milliseconds has passed.
089     *
090     * <p>
091     * A special case of this is effected when {@link PersistentObject#setRoot setRoot()} is never explicitly invoked
092     * by the application. Then some other process must be responsible for all database updates, and this class automatically
093     * picks them up, validates them, and send out notifications.
094     *
095     * <h3>Empty Starts and Stops</h3>
096     *
097     * <p>
098     * An "empty start" occurs when an instance is {@linkplain #start started} but the persistent XML file is either missing,
099     * does not validate, or cannot be read for some other reason. In such cases, the instance will start with no object graph,
100     * and {@link #getRoot} will return null. This situation will correct itself as soon as the object graph is written via
101     * {@link #setRoot setRoot()} or the persistent file appears (effecting an "out-of-band" update). At that time, {@linkplain
102     * #addListener listeners} will be notified for the first time, with {@link PersistentObjectEvent#getOldRoot} returning null.
103     *
104     * <p>
105     * Whether empty starts are allowed is determined by the {@link #isAllowEmptyStart allowEmptyStart} property (default
106     * {@code false}). When empty starts are disallowed and the persistent XML file cannot be successfully read,
107     * then {@link #start} will instead throw an immediate {@link PersistentObjectException}.
108     *
109     * <p>
110     * Similarly, "empty stops" are allowed when the {@link #isAllowEmptyStop allowEmptyStop} property is set to {@code true}
111     * (by default it is {@code false}). An "empty stop" occurs when a null value is passed to {@link #setRoot setRoot()}.
112     * Subsequent invocations of {@link #getRoot} will return {@code null}; however, the persistent file is <b>not</b>
113     * modified when null is passed to {@link #setRoot}. When empty stops are disallowed, then invoking
114     * {@link #setRoot} with a {@code null} object will result in an {@link IllegalArgumentException}.
115     *
116     * <p>
117     * Allowing empty starts and/or stops essentially creates an "unconfigured" state represented by a null root object.
118     * When empty starts and empty stops are both disallowed, there is no "unconfigured" state: {@link #getRoot} can be
119     * relied upon to always return a non-null, validated root object.
120     *
121     * <h3>Shared Roots</h3>
122     *
123     * Each time {@link #setRoot setRoot()} or {@link #getRoot getRoot()} is invoked, a deep copy of the root object
124     * is made. This prevents external code from changing any node in the "official" object graph held by this instance,
125     * and allows invokers of {@link #getRoot getRoot()} to modify to the returned object graph without affecting other invokers.
126     * However, there may be cases where this deep copy is an expensive operation in terms of time or memory.
127     * The {@link #getSharedRoot getSharedRoot()} method can be used in these situations. This method returns the same
128     * root object each time it is invoked (this shared root is itself a deep copy of the "official" root). Therefore, only
129     * the very first invocation pays the price of a copy. However, all invokers of {@link #getSharedRoot getSharedRoot()}
130     * must treat the object graph as read-only to avoid each other seeing unexpected changes.
131     *
132     * <h3>Delegate Function</h3>
133     *
134     * <p>
135     * Instances must be configured with a {@link PersistentObjectDelegate} that knows how to validate the object graph
136     * and perform conversions to and from XML. See {@link PersistentObjectDelegate} and its implementations for details.
137     *
138     * <h3>Schema Changes</h3>
139     *
140     * <p>
141     * Like any database, the XML schema may evolve over time. The {@link PersistentObjectSchemaUpdater} class provides a simple
142     * way to apply and manage schema updates using XSLT transforms.
143     *
144     * @param <T> type of the root persistent object
145     * @see PersistentObjectDelegate
146     * @see PersistentObjectSchemaUpdater
147     */

410        /**
411         * Start this instance. Does nothing if already started.
412         *
413         * @throws PersistentObjectException if an error occurs
414         */
415        public synchronized void start() {
416    
417            // Already started?
418            if (this.started)
419                return;
420    
421            // Sanity check
422            if (this.streamRepository == null)
423                throw new PersistentObjectException("no file configured");
424            if (this.delegate == null)
425                throw new PersistentObjectException("no delegate configured");
426    
427            // Create executor services
428            this.scheduledExecutor = Executors.newSingleThreadScheduledExecutor();
429            this.notifyExecutor = Executors.newSingleThreadExecutor();
430    
431            // Read file (if it exists)
432            this.log.info(this + ": starting");
433            if (this.getFile().exists()) {
434    
435                // Apply file
436                try {
437                    this.applyFile(this.getFile().lastModified(), true);
438                } catch (PersistentObjectException e) {
439                    if (!this.isAllowEmptyStart())
440                        throw e;
441                    this.log.warn("empty start: unable to load persistent file `" + this.getFile() + "': " + e);
442                }
443            } else {
444    
445                // Persistent file does not exist, so get default value from delegate
446                final T defaultValue = this.delegate.getDefaultValue();
447    
448                // If no default value, we have an empty start situation, otherwise apply the default value
449                if (defaultValue == null) {
450                    if (!this.isAllowEmptyStart()) {
451                        throw new PersistentObjectException("persistent file `" + this.getFile() + "' does not exist,"
452                          + " there is no default value, and empty starts are disallowed");
453                    }
454                    this.log.info(this + ": empty start: persistent file `" + this.getFile() + "' does not exist"
455                      + " and there is no default value");
456                } else {
457                    this.log.info(this + ": persistent file `" + this.getFile() + "' does not exist, applying default value");
458                    try {
459                        this.setRootInternal(defaultValue, 0, false, true);
460                    } catch (PersistentObjectException e) {                             // e.g., validation failure
461                        if (!this.isAllowEmptyStart())
462                            throw e;
463                        this.log.warn("empty start: unable to apply default value: " + e);
464                    }
465                }
466            }
467    
468            // Start checking the file
469            if (this.checkInterval > 0) {
470                this.scheduledExecutor.scheduleWithFixedDelay(new Runnable() {
471                    @Override
472                    public void run() {
473                        PersistentObject.this.checkFileTimeout();
474                    }
475                }, this.checkInterval, this.checkInterval, TimeUnit.MILLISECONDS);
476            }
477    
478            // Done
479            this.started = true;
480        }
481    
482        /**
483         * Stop this instance. Does nothing if already stopped.
484         *
485         * @throws PersistentObjectException if a delayed write back is pending and error occurs while performing the write
486         */
487        public synchronized void stop() {
488    
489            // Already stopped?
490            if (!this.started)
491                return;
492    
493            // Perform any lingering pending save now
494            if (this.cancelPendingWrite())
495                this.writeback();
496    
497            // Stop executor services
498            this.log.info(this + ": shutting down");
499            this.scheduledExecutor.shutdown();
500            this.notifyExecutor.shutdown();
501            this.awaitTermination(this.scheduledExecutor, "scheduledExecutor");
502            this.awaitTermination(this.notifyExecutor, "notifyExecutor");
503    
504            // Reset
505            this.scheduledExecutor = null;
506            this.notifyExecutor = null;
507            this.root = null;
508            this.sharedRoot = null;
509            this.writebackRoot = null;
510            this.version = 0;
511            this.timestamp = 0;
512            this.started = false;
513        }

526         * Atomically read the root object.
527         *
528         * <p>
529         * In the situation of an empty start or empty stop, this instance is "unconfigured" and null will be returned.
530         * This can only happen if {@link #setAllowEmptyStart setAllowEmptyStart(true)} or {@link #setAllowEmptyStop
531         * setAllowEmptyStop(true)} has been invoked.
532         *
533         * <p>
534         * This returns a deep copy of the current root object; any subsequent modifications are not written back.
535         * Use {@link #getSharedRoot} instead to avoid the cost of the deep copy at the risk of seeing modifications
536         * caused by other invokers.
537         *
538         * @return the current root instance, or null during an empty start or empty stop
539         * @throws IllegalStateException if this instance is not started
540         * @throws PersistentObjectException if an error occurs
541         */
542        public synchronized T getRoot() {
543    
544            // Sanity check
545            if (!this.started)
546                throw new IllegalStateException("not started");
547    
548            // Copy root
549            return this.root != null ? this.delegate.copy(this.root) : null;
550        }

602        /**
603         * Atomically update the root object.
604         *
605         * <p>
606         * The given object is deep-copied, the copy replaces the current root, and the new version number is returned.
607         * If there is no write delay configured, the new version is written to the underlying file immediately and
608         * a successful return from this method indicates the new root has been persisted. Otherwise, the write will
609         * occur at a later time in a separate thread.
610         * </p>
611         *
612         * <p>
613         * If {@code expectedVersion} is non-zero, then if the current version is not equal to it,
614         * a {@link PersistentObjectVersionException} exception is thrown. This mechanism
615         * can be used for optimistic locking.
616         * </p>
617         *
618         * <p>
619         * If empty stops are allowed, then {@code newRoot} may be null, in which case it replaces the
620         * current root and subsequent calls to {@link #getRoot} will return null. When a null
621         * {@code newRoot} is set, the persistent file is <b>not</b> modified.
622         * </p>
623         *
624         * <p>
625         * If the given root object is {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current
626         * root object, then no action is taken and the current (unchanged) version number is returned.
627         * </p>
628         *
629         * <p>
630         * After a successful change, any registered {@linkplain PersistentObjectListener listeners} are notified in a
631         * separate thread from the one that invoked this method.
632         * </p>
633         *
634         * @param newRoot new persistent object
635         * @param expectedVersion expected current version number, or zero to ignore the current version number
636         * @return the new current version number (unchanged if {@code newRoot} is
637         *  {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current root)
638         * @throws IllegalArgumentException if {@code newRoot} is null and empty stops are disallowed
639         * @throws IllegalArgumentException if {@code expectedVersion} is negative
640         * @throws IllegalStateException if this instance is not started
641         * @throws PersistentObjectException if an error occurs
642         * @throws PersistentObjectVersionException if {@code expectedVersion} is non-zero and not equal to the current version
643         * @throws PersistentObjectValidationException if the new root has validation errors
644         */
645        public final synchronized long setRoot(T newRoot, long expectedVersion) {
646            return this.setRootInternal(newRoot, expectedVersion, false, false);
647        }
648    
649        @SuppressWarnings("unchecked")
650        private synchronized long setRootInternal(T newRoot, long expectedVersion, boolean readingFile, boolean allowNotStarted) {
651    
652            // Sanity check
653            if (newRoot == null && !this.isAllowEmptyStop())
654                throw new IllegalArgumentException("newRoot is null but empty stops are not enabled");
655            if (!this.started && !allowNotStarted)
656                throw new IllegalStateException("not started");
657            if (expectedVersion < 0)
658                throw new IllegalStateException("negative expectedVersion");
659    
660            // Check version number
661            if (expectedVersion != 0 && this.version != expectedVersion)
662                throw new PersistentObjectVersionException(this.version, expectedVersion);
663    
664            // Check for sameness
665            if (this.root == null && newRoot == null)
666                return this.version;
667            if (this.root != null && newRoot != null && this.delegate.isSameGraph(this.root, newRoot))
668                return this.version;
669    
670            // Validate the new root
671            if (newRoot != null) {
672                Set<ConstraintViolation<T>> violations = this.delegate.validate(newRoot);
673                if (!violations.isEmpty())
674                    throw new PersistentObjectValidationException((Set<ConstraintViolation<?>>)(Object)violations);
675            }
676    
677            // Do the update
678            final T oldRoot = this.root;
679            this.root = newRoot != null ? this.delegate.copy(newRoot) : null;
680            this.version++;
681            this.sharedRoot = null;
682    
683            // Perform write-back, either now or later
684            if (!readingFile && this.root != null) {
685                this.writebackRoot = this.root;
686                if (this.writeDelay == 0)
687                    this.writeback();
688                else if (this.pendingWrite == null) {
689                    this.pendingWrite = this.scheduledExecutor.schedule(new Runnable() {
690                        @Override
691                        public void run() {
692                            PersistentObject.this.writeTimeout();
693                        }
694                    }, this.writeDelay, TimeUnit.MILLISECONDS);
695                }
696            }
697    
698            // Notify listeners
699            this.notifyListeners(this.version, oldRoot, newRoot);
700    
701            // Done
702            return this.version;
703        }

709         * The is a convenience method, equivalent to:
710         *  <blockquote>

712         *  </blockquote>
713         *
714         * <p>
715         * This method cannot throw {@link PersistentObjectVersionException}.

721        /**
722         * Check the persistent file for an "out-of-band" update.
723         *
724         * <p>
725         * If the persistent file has a newer timestamp than the timestamp of the most recently read
726         * or written version, then it will be read and applied to this instance.
727         *
728         * @throws IllegalStateException if this instance is not started
729         * @throws PersistentObjectException if an error occurs
730         */
731        public synchronized void checkFile() {
732    
733            // Sanity check
734            if (!this.started)
735                throw new IllegalStateException("not started");
736    
737            // Get file timestamp
738            long fileTime = this.getFile().lastModified();
739            if (fileTime == 0)
740                return;
741    
742            // Check whether file has newly appeared or just been updated
743            if (this.timestamp != 0 && fileTime <= this.timestamp)
744                return;
745    
746            // Read new file
747            this.log.info(this + ": detected out-of-band update of persistent file `" + this.getFile() + "'");
748            this.applyFile(fileTime, false);
749        }

784        /**
785         * Read the persistent file.
786         *
787         * @throws PersistentObjectException if an error occurs
788         */
789        protected T read() {
790    
791            // Open file
792            this.log.info(this + ": reading persistent file `" + this.getFile() + "'");
793            BufferedInputStream input;
794            try {
795                input = new BufferedInputStream(this.streamRepository.getInputStream());
796            } catch (IOException e) {
797                throw new PersistentObjectException("error opening persistent file", e);
798            }
799    
800            // Parse XML
801            T obj;
802            try {
803                StreamSource source = new StreamSource(input);
804                source.setSystemId(this.getFile());
805                try {
806                    obj = this.delegate.deserialize(source);
807                } catch (IOException e) {
808                    throw new PersistentObjectException("error reading persistent file", e);
809                }
810            } finally {
811                try {
812                    input.close();
813                } catch (IOException e) {
814                    // ignore
815                }
816            }
817    
818            // Check result
819            if (obj == null)
820                throw new PersistentObjectException("null object returned by delegate.deserialize()");
821    
822            // Done
823            return obj;
824        }

826        /**
827         * Write the persistent file and rotate any backups.
828         *
829         * <p>
830         * A temporary file is created in the same directory and then renamed to provide for an atomic update
831         * (on supporting operating systems).
832         *
833         * @throws IllegalArgumentException if {@code obj} is null
834         * @throws PersistentObjectException if an error occurs
835         */
836        protected final synchronized void write(T obj) {
837    
838            // Sanity check
839            if (obj == null)
840                throw new IllegalArgumentException("null obj");
841    
842            // Open atomic update output and buffer it
843            AtomicUpdateFileOutputStream updateOutput;
844            try {
845                updateOutput = this.streamRepository.getOutputStream();
846            } catch (IOException e) {
847                throw new PersistentObjectException("error creating temporary file", e);
848            }
849            BufferedOutputStream output = new BufferedOutputStream(updateOutput);
850    
851            // Serialize to XML
852            try {
853    
854                // Set up XML result
855                Result result = this.createResult(output, updateOutput.getTempFile());
856    
857                // Serialize root object
858                try {
859                    this.delegate.serialize(obj, result);
860                } catch (IOException e) {
861                    throw new PersistentObjectException("error writing persistent file", e);
862                }
863    
864                // Commit output
865                try {
866                    output.close();
867                } catch (IOException e) {
868                    throw new PersistentObjectException("error closing temporary file", e);
869                }
870    
871                // Success
872                output = null;
873            } finally {
874                if (output != null)
875                    updateOutput.cancel();
876            }
877    
878            // Update file timestamp
879            this.timestamp = this.streamRepository.getFileTimestamp();
880        }

882        /**
883         * Create a {@link Result} targeting the given {@link OutputStream}.
884         *
885         * <p>
886         * The implementation in {@link PersistentObject} creates and returns a {@link StreamResult}.
887         * </p>
888         *
889         * @param output XML output stream
890         * @param systemId system ID
891         */
892        protected Result createResult(OutputStream output, File systemId) {
893            StreamResult result = new StreamResult(output);
894            result.setSystemId(systemId);
895            return result;
896        }

898        /**
899         * Notify listeners of a change in value.
900         *
901         * @param newVersion the version number associated with the new root
902         */
903        protected void notifyListeners(long newVersion, T oldRoot, T newRoot) {
904    
905            // Snapshot listeners
906            final ArrayList<PersistentObjectListener<T>> listenersCopy = new ArrayList<PersistentObjectListener<T>>();
907            synchronized (this.listeners) {
908                listenersCopy.addAll(this.listeners);
909            }
910    
911            // Notify them
912            final PersistentObjectEvent<T> event = new PersistentObjectEvent<T>(this, newVersion, oldRoot, newRoot);
913            this.notifyExecutor.submit(new Runnable() {
914                @Override
915                public void run() {
916                    PersistentObject.this.doNotifyListeners(listenersCopy, event);
917                }
918            });
919        }

921        // Read the persistent file and apply it
922        private synchronized void applyFile(long newTimestamp, boolean allowNotStarted) {
923            this.cancelPendingWrite();
924            this.timestamp = newTimestamp;                              // update timestamp even if update fails to avoid loops
925            this.setRootInternal(this.read(), 0, true, allowNotStarted);
926        }

928        // Handle a write-back timeout
929        private synchronized void writeTimeout() {
930    
931            // Check for cancel race
932            if (this.pendingWrite == null)
933                return;
934            this.pendingWrite = null;
935    
936            // Write it
937            try {
938                this.writeback();
939            } catch (ThreadDeath t) {
940                throw t;
941            } catch (Throwable t) {
942                this.delegate.handleWritebackException(this, t);
943            }
944        }
945    
946        // Handle a check file timeout
947        private synchronized void checkFileTimeout() {
948    
949            // Handle race condition
950            if (!this.started)
951                return;
952    
953            // Check file
954            try {
955                this.checkFile();
956            } catch (ThreadDeath t) {
957                throw t;
958            } catch (Throwable t) {
959                this.log.error(this + ": error attempting to apply out-of-band update", t);
960            }
961        }
962    
963        // Write back root to persistent file
964        private synchronized void writeback() {
965            T objectToWrite = this.writebackRoot;
966            assert objectToWrite != null;
967            this.writebackRoot = null;
968            this.write(objectToWrite);
969        }
970    
971        // Cancel a pending write and return true if there was one
972        private synchronized boolean cancelPendingWrite() {
973            if (this.pendingWrite == null)
974                return false;
975            this.pendingWrite.cancel(false);
976            this.pendingWrite = null;
977            return true;
978        }
979    
980        // Notify listeners. This is invoked in a separate thread.
981        private void doNotifyListeners(ArrayList<PersistentObjectListener<T>> list, PersistentObjectEvent<T> event) {
982            for (PersistentObjectListener<T> listener : list) {
983                try {
984                    listener.handleEvent(event);
985                } catch (ThreadDeath t) {
986                    throw t;
987                } catch (Throwable t) {
988                    this.log.error(this + ": error notifying listener " + listener, t);
989                }
990            }
991        }

993        // Wait for an ExecutorService to completely shut down
994        private void awaitTermination(ExecutorService executor, String name) {
995            boolean shutdown = false;
996            try {
997                shutdown = executor.awaitTermination(EXECUTOR_SHUTDOWN_TIMEOUT, TimeUnit.MILLISECONDS);
998            } catch (InterruptedException e) {
999                this.log.warn(this + ": interrupted while awaiting " + name + " termination");
1000                Thread.currentThread().interrupt();
1001            }
1002            if (!shutdown)
1003                this.log.warn(this + ": failed to completely shut down " + name);
1004        }
