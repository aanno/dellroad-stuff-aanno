033    /**
034     * Main class for Simple XML Persistence Objects (POBJ).
035     *
036     * <h3>Overview</h3>
037     *
038     * <p>
039     * Instances model an in-memory "database" represented by a root Java object and the graph of other objects that
040     * it references. The object graph is backed by a persistent XML file, which is read at initialization time and
041     * re-written after each change.
042     *
043     * <p>
044     * Changes are applied "wholesale" to the entire object graph, and are serialized and atomic. In other words, the
045     * entire object graph is read from, and written to, this class by value. As a result, it is not possible to change
046     * only a portion of the "database". The entire object graph is read and written as one thing. Similarly, the
047     * persistent XML file is updated by writing out a new, temporary copy and renaming the copy onto the original,
048     * using {@link File#renameTo File.renameTo()} for atomicity (on systems that support it, e.g., UNIX variants).
049     *
050     * <h3>Update Details</h3>
051     *
052     * <p>
053     * When the object graph is updated, it must pass validation checks, and then the persistent XML file is updated and
054     * listener notifications are sent out. Support for delayed write-back of the persistent XML file is included: this
055     * allows modifications that occur in rapid succession to be consolidated into a single filesystem write operation.
056     *
057     * <p>
058     * Support for optimistic locking is included. There is a "current version" number which is incremented each
059     * time the object graph is updated; writes may optionally specify this number to ensure no intervening changes
060     * have occurred. If concurrent updates are expected, applications may choose to implement a 3-way merge algorithm
061     * of some kind to handle optimistic locking failures.
062     *
063     * <p>
064     * Instances can be configured to preserve one or more backup copies of the persistent file on systems that support
065     * hard links (requires <a href="https://github.com/twall/jna">JNA</a>; see {@link HardLink}).
066     * Set the {@link #getNumBackups numBackups} property to enable.
067     *
068     * <h3>"Out-of-band" Writes</h3>
069     *
070     * <p>
071     * When a non-zero {@linkplain #getCheckInterval check interval} is configured, instances support "out-of-band" writes
072     * to the XML persistent file by some other process. This can be handy in cases where the other process (perhaps hand edits)
073     * is updating the persistent file and you want to have a running process pick up the changes just as if
074     * {@link PersistentObject#setRoot setRoot()} had been invoked. In particular, instances will detect the appearance
075     * of a new persistent file after an instance has started without one. In all cases, persistent objects must properly validate.
076     *
077     * <p>
078     * A special case of this is effected when {@link PersistentObject#setRoot setRoot()} is never explicitly invoked
079     * by the application. Then some other process must be responsible for all database updates, and this class automatically
080     * picks them up, validates them, and send out notifications.
081     *
082     * <h3>Empty Starts</h3>
083     *
084     * <p>
085     * An "empty start" occurs when an instance is {@linkplain #start started} but the persistent XML file is either missing,
086     * does not validate, or cannot be read for some other reason. In such cases, the instance will start with no object graph,
087     * and {@link #getRoot} will initially return null. This situation will correct itself as soon as the object graph is written
088     * via {@link #setRoot setRoot()} or the persistent file appears (effecting an "out-of-band" update).
089     *
090     * <p>
091     * Whther empty starts are allowed is determined by the {@link #isAllowEmptyStart allowEmptyStart} property (default
092     * {@code false}). When empty starts are disallowed, then {@link #start} will instead throw a {@link PersistentObjectException}.
093     * In this configuration, {@link #getRoot} can be relied upon to always return a non-null, valid root.
094     *
095     * <h3>Delegate Function</h3>
096     *
097     * <p>
098     * Instances must be configured with a {@link PersistentObjectDelegate} that knows how to validate the object graph
099     * and perform conversions to and from XML. See {@link PersistentObjectDelegate} and its implementations for details.
100     *
101     * <h3>Schema Changes</h3>
102     *
103     * <p>
104     * Like any database, the XML schema may evolve over time. The {@link PersistentObjectSchemaUpdater} class provides a simple
105     * way to apply and manage schema updates using XSLT transforms.
106     *
107     * @param <T> type of the root persistent object
108     * @see PersistentObjectDelegate
109     */
110    public class PersistentObject<T> {

251        /**
252         * Start this instance. Does nothing if already started.
253         *
254         * @throws PersistentObjectException if an error occurs
255         */
256        public synchronized void start() {
257    
258            // Already started?
259            if (this.started)
260                return;
261    
262            // Create executor services
263            this.scheduledExecutor = Executors.newSingleThreadScheduledExecutor();
264            this.notifyExecutor = Executors.newSingleThreadExecutor();
265    
266            // Read file (if it exists)
267            this.log.info(this + ": starting");
268            if (this.persistentFile.exists()) {
269                try {
270                    this.applyFile(this.persistentFile.lastModified());
271                } catch (PersistentObjectException e) {
272                    if (!this.isAllowEmptyStart())
273                        throw e;
274                    this.log.warn("empty start: unable to load persistent file `" + this.persistentFile + "': " + e);
275                }
276            } else {
277                if (!this.isAllowEmptyStart())
278                    throw new PersistentObjectException("persistent file `" + this.persistentFile + "' does not exist");
279                this.log.info(this + ": empty start: persistent file `" + this.persistentFile + "' does not exist");
280            }
281    
282            // Start checking the file
283            if (this.checkInterval > 0) {
284                this.scheduledExecutor.scheduleWithFixedDelay(new Runnable() {
285                    @Override
286                    public void run() {
287                        PersistentObject.this.checkFileTimeout();
288                    }
289                }, this.checkInterval, this.checkInterval, TimeUnit.MILLISECONDS);
290            }
291    
292            // Done
293            this.started = true;
294        }
295    
296        /**
297         * Stop this instance. Does nothing if already stopped.
298         *
299         * @throws PersistentObjectException if a delayed write back is pending and error occurs while performing the write
300         */
301        public synchronized void stop() {
302    
303            // Already stopped?
304            if (!this.started)
305                return;
306    
307            // Perform any lingering pending save now
308            if (this.cancelPendingWrite())
309                this.write(this.root);
310    
311            // Stop executor services
312            this.log.info(this + ": shutting down");
313            this.scheduledExecutor.shutdown();
314            this.notifyExecutor.shutdown();
315            this.awaitTermination(this.scheduledExecutor, "scheduledExecutor");
316            this.awaitTermination(this.notifyExecutor, "notifyExecutor");
317    
318            // Reset
319            this.scheduledExecutor = null;
320            this.notifyExecutor = null;
321            this.root = null;
322            this.version = 0;
323            this.timestamp = 0;
324            this.started = false;
325        }

327        /**
328         * Atomically read the root object.
329         *
330         * <p>
331         * If there is no persistent file and no value has been set, null will be returned.
332         * However the persistent file may appear "out of band" at any time; if so, this will
333         * be detected within the configured {@linkplain #getCheckInterval check interval}.
334         *
335         * <p>
336         * This returns a deep copy of the current root object; any subsequent modifications are not written back.
337         *
338         * @return the current root instance, or null if after an "empty start"
339         * @throws IllegalStateException if this instance is not started
340         * @throws PersistentObjectException if an error occurs
341         */
342        public synchronized T getRoot() {
343    
344            // Sanity check
345            if (!this.started)
346                throw new IllegalStateException("not started");
347    
348            // Copy root
349            return this.root != null ? this.delegate.copy(this.root) : null;
350        }

357         * This method is more efficient than {@link #getRoot}, but all callers must agree not to modify the returned object
358         * or any object in its graph of references.
359         *
360         * @return shared copy of the root instance, or null if after an "empty start"

363            if (this.sharedRoot == null)
364                this.sharedRoot = this.getRoot();
365            return this.sharedRoot;

368        /**
369         * Atomically update the root object.
370         *
371         * <p>
372         * The given object is deep-copied and the copy replaces the current root.
373         *
374         * <p>
375         * If {@code expectedVersion} is non-zero, then if the current version is not equal to it,
376         * a {@link PersistentObjectVersionException} exception is thrown. This mechanism
377         * can be used for optimistic locking.
378         *
379         * @param newRoot new persistent object
380         * @param expectedVersion expected current version number, or zero to ignore the current version number
381         * @throws IllegalArgumentException if {@code newRoot} is null
382         * @throws IllegalArgumentException if {@code version} is negative
383         * @throws IllegalStateException if this instance is not started
384         * @throws PersistentObjectException if an error occurs
385         * @throws PersistentObjectVersionException if {@code version} is non-zero and not equal to the current version
386         * @throws PersistentObjectValidationException if the new root has validation errors
387         */
388        public final synchronized void setRoot(T newRoot, long expectedVersion) {
389            this.setRootInternal(newRoot, expectedVersion, false);
390        }

392        @SuppressWarnings("unchecked")
393        private synchronized void setRootInternal(T newRoot, long expectedVersion, boolean readingFile) {
394    
395            // Sanity check
396            if (newRoot == null)
397                throw new IllegalArgumentException("null newRoot");
398            if (!this.started && !readingFile)
399                throw new IllegalStateException("not started");
400            if (expectedVersion < 0)
401                throw new IllegalStateException("negative expectedVersion");
402    
403            // Check version number
404            if (expectedVersion != 0 && this.version != expectedVersion)
405                throw new PersistentObjectVersionException(this.version, expectedVersion);
406    
407            // Check for sameness
408            if (this.root != null && this.delegate.isSameGraph(this.root, newRoot))
409                return;
410    
411            // Validate the new root
412            Set<ConstraintViolation<T>> violations = this.delegate.validate(newRoot);
413            if (!violations.isEmpty())
414                throw new PersistentObjectValidationException((Set<ConstraintViolation<?>>)(Object)violations);
415    
416            // Do the update
417            final T oldRoot = this.root;
418            this.root = this.delegate.copy(newRoot);
419            this.version++;
420            this.sharedRoot = null;
421    
422            // Perform write-back, either now or later
423            if (!readingFile) {
424                if (this.writeDelay == 0)
425                    this.write(this.root);
426                else if (this.pendingWrite == null) {
427                    this.pendingWrite = this.scheduledExecutor.schedule(new Runnable() {
428                        @Override
429                        public void run() {
430                            PersistentObject.this.writeTimeout();
431                        }
432                    }, this.writeDelay, TimeUnit.MILLISECONDS);
433                }
434            }
435    
436            // Notify listeners
437            this.notifyListeners(this.version, oldRoot, newRoot);
438        }

456        /**
457         * Check the persistent file for an "out-of-band" update.
458         *
459         * <p>
460         * If the persistent file has a newer timestamp than the timestamp of the most recently read
461         * or written version, then it will be read and applied to this instance.
462         *
463         * @throws IllegalStateException if this instance is not started
464         * @throws PersistentObjectException if an error occurs
465         */
466        public synchronized void checkFile() {
467    
468            // Sanity check
469            if (!this.started)
470                throw new IllegalStateException("not started");
471    
472            // Get file timestamp
473            long fileTime = this.persistentFile.lastModified();
474            if (fileTime == 0)
475                return;
476    
477            // Check whether file has newly appeared or just been updated
478            if (this.timestamp != 0 && fileTime <= this.timestamp)
479                return;
480    
481            // Read new file
482            this.log.info(this + ": detected out-of-band update of persistent file `" + this.persistentFile + "'");
483            this.applyFile(fileTime);
484        }

523        /**
524         * Read the persistent file.
525         *
526         * @throws PersistentObjectException if an error occurs
527         */
528        protected T read() {
529    
530            // Open file
531            this.log.info(this + ": reading persistent file `" + this.persistentFile + "'");
532            BufferedInputStream input;
533            try {
534                input = new BufferedInputStream(new FileInputStream(this.persistentFile));
535            } catch (IOException e) {
536                throw new PersistentObjectException("error opening persistent file", e);
537            }
538    
539            // Parse XML
540            T obj;
541            try {
542                StreamSource source = new StreamSource(input);
543                source.setSystemId(this.persistentFile);
544                try {
545                    obj = this.delegate.deserialize(source);
546                } catch (IOException e) {
547                    throw new PersistentObjectException("error reading persistent file", e);
548                }
549            } finally {
550                try {
551                    input.close();
552                } catch (IOException e) {
553                    // ignore
554                }
555            }
556    
557            // Check result
558            if (obj == null)
559                throw new PersistentObjectException("null object returned by delegate.deserialize()");
560    
561            // Done
562            return obj;
563        }

565        /**
566         * Write the persistent file and rotate any backups.
567         *
568         * <p>
569         * A temporary file is created in the same directory and then renamed to provide for an atomic update
570         * (on supporting operating systems).
571         *
572         * @throws IllegalArgumentException if {@code obj} is null
573         * @throws PersistentObjectException if an error occurs
574         */
575        protected final synchronized void write(T obj) {
576    
577            // Sanity check
578            if (obj == null)
579                throw new IllegalArgumentException("null obj");
580    
581            // Create temporary file
582            this.log.info(this + ": writing persistent file `" + this.persistentFile + "'");
583            File tempFile;
584            try {
585                tempFile = File.createTempFile(this.persistentFile.getName(), null, this.persistentFile.getParentFile());
586            } catch (IOException e) {
587                throw new PersistentObjectException("error creating temporary file", e);
588            }
589            try {
590    
591                // Open temporary file
592                BufferedOutputStream output;
593                try {
594                    output = new BufferedOutputStream(new FileOutputStream(tempFile));
595                } catch (IOException e) {
596                    throw new PersistentObjectException("error opening to temporary file", e);
597                }
598    
599                // Serialize to XML
600                try {
601                    StreamResult result = new StreamResult(output);
602                    result.setSystemId(tempFile);
603                    try {
604                        this.delegate.serialize(obj, result);
605                    } catch (IOException e) {
606                        throw new PersistentObjectException("error writing persistent file", e);
607                    }
608                    try {
609                        output.close();
610                    } catch (IOException e) {
611                        throw new PersistentObjectException("error closing temporary file", e);
612                    }
613                    output = null;
614                } finally {
615                    try {
616                        if (output != null)
617                            output.close();
618                    } catch (IOException e) {
619                        // ignore
620                    }
621                }
622    
623                // Get new modification time (prior to the rename, to avoid a race condition)
624                long newTimestamp = tempFile.lastModified();
625    
626                // Rotate backups
627                for (int i = this.numBackups - 1; i >= 0; i--) {
628                    File src = i > 0 ? new File(this.persistentFile.toString() + "." + i) : this.persistentFile;
629                    File dest = new File(this.persistentFile.toString() + "." + (i + 1));
630                    if (i == 0) {
631                        try {
632                            HardLink.link(src, dest);
633                        } catch (IOException e) {
634                            this.log.warn("unable to backup persistent file to `" + dest + "': " + e);
635                        }
636                    } else
637                        src.renameTo(dest);
638                }
639    
640                // Rename file
641                if (!tempFile.renameTo(this.persistentFile)) {
642                    throw new PersistentObjectException("error renaming temporary file `"
643                      + tempFile.getName() + "' to `" + this.persistentFile.getName() + "'");
644                }
645                tempFile = null;
646    
647                // Update file timestamp
648                this.timestamp = newTimestamp;
649            } finally {
650                if (tempFile != null)
651                    tempFile.delete();
652            }
653        }

655        /**
656         * Notify listeners of a change in value.
657         *
658         * @param newVersion the version number associated with the new root
659         */
660        protected void notifyListeners(long newVersion, T oldRoot, T newRoot) {
661    
662            // Snapshot listeners
663            final ArrayList<PersistentObjectListener<T>> listenersCopy = new ArrayList<PersistentObjectListener<T>>();
664            synchronized (this.listeners) {
665                listenersCopy.addAll(this.listeners);
666            }
667    
668            // Notify them
669            final PersistentObjectEvent<T> event = new PersistentObjectEvent<T>(this, newVersion, oldRoot, newRoot);
670            this.notifyExecutor.submit(new Runnable() {
671                @Override
672                public void run() {
673                    PersistentObject.this.doNotifyListeners(listenersCopy, event);
674                }
675            });
676        }

678        // Read the persistent file and apply it
679        private synchronized void applyFile(long newTimestamp) {
680            this.cancelPendingWrite();
681            this.timestamp = newTimestamp;                              // update timestamp even if update fails to avoid loops
682            this.setRootInternal(this.read(), 0, true);
683        }
684    
685        // Handle a write-back timeout
686        private synchronized void writeTimeout() {
687    
688            // Check for cancel race
689            if (this.pendingWrite == null)
690                return;
691            this.pendingWrite = null;
692    
693            // Write it
694            try {
695                this.write(this.root);
696            } catch (ThreadDeath t) {
697                throw t;
698            } catch (Throwable t) {
699                this.delegate.handleWritebackException(this, t);
700            }
701        }
702    
703        // Handle a check file timeout
704        private synchronized void checkFileTimeout() {
705    
706            // Handle race condition
707            if (!this.started)
708                return;
709    
710            // Check file
711            try {
712                this.checkFile();
713            } catch (ThreadDeath t) {
714                throw t;
715            } catch (Throwable t) {
716                this.log.error(this + ": error attempting to apply out-of-band update", t);
717            }
718        }
719    
720        // Cancel a pending write and return true if there was one
721        private synchronized boolean cancelPendingWrite() {
722            if (this.pendingWrite == null)
723                return false;
724            this.pendingWrite.cancel(false);
725            this.pendingWrite = null;
726            return true;
727        }
728    
729        // Notify listeners. This is invoked in a separate thread.
730        private void doNotifyListeners(ArrayList<PersistentObjectListener<T>> list, PersistentObjectEvent<T> event) {
731            for (PersistentObjectListener<T> listener : list) {
732                try {
733                    listener.handleEvent(event);
734                } catch (ThreadDeath t) {
735                    throw t;
736                } catch (Throwable t) {
737                    this.log.error(this + ": error notifying listener " + listener, t);
738                }
739            }
740        }
741    
742        // Wait for an ExecutorService to completely shut down
743        private void awaitTermination(ExecutorService executor, String name) {
744            boolean shutdown = false;
745            try {
746                shutdown = executor.awaitTermination(EXECUTOR_SHUTDOWN_TIMEOUT, TimeUnit.MILLISECONDS);
747            } catch (InterruptedException e) {
748                this.log.warn(this + ": interrupted while awaiting " + name + " termination");
749            }
750            if (!shutdown)
751                this.log.warn(this + ": failed to completely shut down " + name);
752        }
